Json_value_type

On this page, we will explain how works the json_value_type.

enum json_value_type

typedef enum {
    NONE,
    BOOLEAN,
    INT,
    STRING,
    OBJECT,
    ARRAY
} json_value_type;

The json_value_type is the type of value you're getting from the json file. Dont use NONE.
